# Inheritance between interfaces

KtorGen supports inheritance between interfaces with the [delegation pattern](https://kotlinlang.org/docs/delegation.html).

For KtorGen perspective, it's just an interface with a delegated property, doesn't matter how it's implemented,
definition or inheritance parameters; doesn't **check anything**.

## Example

Your base interface for all routes:

```kotlin
sealed interface ApiRoutes {
  val httpClient: HttpClient
  val basePath: String

  @GET
  suspend fun getInfo(): String
}
```

An interface for user routes:

```kotlin
interface UserRoutes : ApiRoutes {
  @GET("{userId}")
  suspend fun getUser(@Path("userId") userId: Int): User
}
```

# Package io.github.kingg22.ktorgen.core

## Core Annotations
This section documents the core annotations used to define options and how the compiler interprets them. These annotations control how Ktorgen scans interfaces, generates client factories, and connects them to Ktor’s HTTP client.

More information, see [Core Annotations](https://kingg22.github.io/ktorgen/annotations/core.html).

# Package io.github.kingg22.ktorgen.http

## HTTP Annotations
These annotations define how Ktorgen maps Kotlin interfaces and methods into HTTP requests for Ktor Client.

Ktorgen’s HTTP annotations are inspired by Retrofit and Ktorfit, providing a familiar developer experience while maintaining KMP compatibility and type-safety.

More information, see [HTTP Annotations](https://kingg22.github.io/ktorgen/annotations/http.html).
